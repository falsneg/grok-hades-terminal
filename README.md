# Hades Terminal

A browser extension that injects a terminal overlay into the Grok sandbox, allowing you to execute commands directly within the web interface.

## Privacy Policy

This extension does not collect, store, or transmit any user data. 

Host permissions (`*://*.grok.com/*`) are required to render the terminal UI on the Grok website. The only data transmitted are your explicit terminal commands, which are sent directly to Grok's servers for execution.
