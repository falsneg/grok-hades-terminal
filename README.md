# Hades Terminal

A UI wrapper that injects a terminal overlay into the Grok sandbox.

## Installation

[falsneg.github.io/grok-hades-terminal](https://falsneg.github.io/grok-hades-terminal/)

## Privacy

The extension does not collect or store user data. It requires `*://*.grok.com/*` host permissions to render the terminal. The only data transmitted are your terminal commands, which are sent directly to Grok's servers for execution.
